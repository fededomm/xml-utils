# XML-utils 
is a cli tool wich wrap some python function to analize a xml document

## Prerequisites
have python3.11 or more, and poetry installed 

## Install 

```sh
    poetry install 
    
    poetry shell 
```

## Usage 
```bash 
    xml-utils --help 
    xml-utils <command> --help
```
